# Glossar — verbindliche Wiedergaben (Konsistenz über alle 16 Kapitel)

Format: griechisch → **l1** (wörtlich) | **l2** (mittel) | **l3** (frei).
Im Zweifel kontextsensibel, aber diese Vorgaben sind die Standardwahl.

## Kernbegriffe
- εὐαγγέλιον → l1 „Evangelium" | l2 „Evangelium / gute Botschaft" | l3 „die gute Nachricht"
- βασιλεία τοῦ θεοῦ → l1 „Reich Gottes" | l2 „Reich Gottes" | l3 „Gottes neue Welt" (bei Erstnennung im Abschnitt ggf. „Gottes neue Welt (das Reich Gottes)")
- μετάνοια → l1 „Umkehr/Buße" | l2 „Umkehr" | l3 „umdenken / das Leben ändern"
- μετανοέω → l1 „Buße tun / umdenken" | l2 „umkehren" | l3 „das Leben ändern, neu anfangen"
- ἄφεσις ἁμαρτιῶν → l1 „Vergebung der Sünden" | l2 „Vergebung der Sünden" | l3 „Vergebung der Schuld"
- ἁμαρτία → l1 „Sünde" | l2 „Sünde" | l3 „Schuld / das Falsche"
- πίστις → „Glaube" (alle); πιστεύω → „glauben"
- ἀμὴν λέγω ὑμῖν → l1 „Amen, ich sage euch" | l2 „Wahrlich, ich sage euch" | l3 „Eins ist sicher: …" / „Ich sage euch ganz klar:"
- ὁ υἱὸς τοῦ ἀνθρώπου → l1 „der Sohn des Menschen" | l2 „der Menschensohn" | l3 „der Menschensohn"
- υἱὸς τοῦ θεοῦ → l1/l2 „Sohn Gottes" | l3 „Gottes Sohn"
- κύριος → „Herr" (alle)
- διδάσκαλος → l1/l2 „Lehrer" | l3 „Lehrer" (Anrede ggf. „Rabbi/Lehrer")
- ῥαββί → „Rabbi"
- γραμματεύς/γραμματεῖς → „Schriftgelehrte(r)" (alle); l3 darf „die Schriftgelehrten (die Gesetzeslehrer)"
- Φαρισαῖοι → „Pharisäer"; Σαδδουκαῖοι → „Sadduzäer"; Ἡρῳδιανοί → „Anhänger des Herodes / Herodianer"
- ἀρχιερεύς → „Hoherpriester"; πρεσβύτεροι → „Älteste"
- συναγωγή → „Synagoge"; τὸ ἱερόν → „der Tempel"; ἡ συνέδριον → „der Hohe Rat / das Synedrium"
- παραβολή → l1/l2 „Gleichnis" | l3 „ein Bild / eine Beispielgeschichte"
- ὄχλος → l1 „Menge" | l2 „Volksmenge / die Leute" | l3 „die vielen Menschen / die Leute"
- οἱ μαθηταί → „die Jünger" (l3 ggf. „seine Jünger / seine Schüler")
- πνεῦμα ἅγιον → „Heiliger Geist"
- πνεῦμα ἀκάθαρτον → l1 „unreiner Geist" | l2 „unreiner Geist" | l3 „böser Geist"
- δαιμόνιον → „Dämon" (l3 ggf. „böser Geist")
- σάββατον → „Sabbat"
- νόμος → „Gesetz"; ἐντολή → „Gebot"
- δοξάζω τὸν θεόν → „Gott preisen / loben"
- σκανδαλίζω → l1 „Anstoß nehmen / zu Fall kommen" | l2 „Anstoß nehmen / abfallen" | l3 „den Glauben verlieren / aufgeben"

## Markus-typische Formeln
- εὐθύς / εὐθέως (sehr häufig!) → l1 „sogleich" | l2 „sofort / sogleich" | l3 „sofort / gleich darauf / da"
- καὶ ἐγένετο / ἐγένετο → l1 „und es geschah / es geschah" | l2 „es geschah / da" | l3 frei auflösen („Da …", „Eines Tages …")
- ἀποκριθεὶς εἶπεν / λέγει → l1 „antwortend sprach/sagt er" | l2 „antwortete und sagte" | l3 „antwortete"
- ἰδού → l1 „siehe" | l2 „siehe / sieh" | l3 „pass auf / schau"
- historisches Präsens (λέγει, ἔρχεται …) → l1 Präsens beibehalten | l2/l3 Präteritum erlaubt
- ἤρξατο + Infinitiv → l1 „er begann zu …" | l2 „er fing an zu …" | l3 oft einfach Haupthandlung
- ἀμήν → siehe oben

## Maße/Begriffe
- δηνάριον → „Denar" (l3 ggf. „ein Tageslohn (Denar)")
- λεπτόν → „Lepton (kleinste Münze)" / l3 „zwei kleinste Münzen"
- κορβᾶν → „Korban (Opfergabe)"
- ταλιθα κουμ → beibehalten „Talita kum" + im Satz die Bedeutung (wie im Text V. erklärt)
- ραββουνι/αββα/ελωι ελωι → aramäische Worte beibehalten, Bedeutung wie im Text

## Weitere NT-Begriffe (für Matthäus, Lukas, Johannes)
- λόγος (Joh 1) → l1/l2 „das Wort" | l3 „das Wort (Gottes)"
- ἐγώ εἰμι (Joh) → l1 „Ich bin" | l2 „Ich bin" | l3 „Ich bin es / Ich bin" (kontextabhängig)
- ζωὴ αἰώνιος → „ewiges Leben" (alle)
- φῶς → „Licht"; σκοτία/σκότος → „Finsternis"; ἀλήθεια → „Wahrheit"
- κόσμος → „Welt"; δόξα → l1/l2 „Herrlichkeit" | l3 „Herrlichkeit/Glanz"
- χάρις → „Gnade"; ἀγάπη → „Liebe"; μαρτυρία → „Zeugnis"
- βασιλεία τῶν οὐρανῶν (Mt) → l1/l2 „Himmelreich" | l3 „Gottes neue Welt (Himmelreich)"
- ὁ πατὴρ ὁ ἐν τοῖς οὐρανοῖς → „(euer/der) Vater im Himmel"
- μακάριος (Seligpreisungen, Mt 5) → l1/l2 „selig" | l3 „glücklich" / „gut hat es, wer …"
- ὑποκριτής → l1/l2 „Heuchler" | l3 „Heuchler / wer nur so tut"

- σταυρόω → „kreuzigen"; ἀνίστημι/ἐγείρω (Auferstehung) → „auferwecken / auferstehen"
