# Glossar — Altes Testament (verbindliche Wiedergaben für Konsistenz)

Gleiche hebräische Wurzel möglichst gleich übersetzen (besonders in l1). l3 darf je Kontext variieren, bleibt aber treu.

## Gottesnamen / Anrede
- יהוה (JHWH) → **der HERR** (beide Fassungen; Kapitälchen-Darstellung in der App)
- אֱלֹהִים (Elohim) → Gott
- אֲדֹנָי (Adonai) → Herr
- אֵל / אֱלוֹהַּ → Gott; שַׁדַּי → der Allmächtige; עֶלְיוֹן → der Höchste
- צְבָאוֹת → (der HERR) der Heerscharen

## Liturgisch / Formeln
- סֶלָה (Sela) → Sela (unübersetzt)
- הַלְלוּ יָהּ → Halleluja
- אָמֵן → Amen
- אַשְׁרֵי → glücklich / selig (l1 „Selig/Glücklich")
- בָּרוּךְ → gepriesen / gesegnet; בָּרַךְ → segnen / preisen

## Zentrale Begriffe
- חֶסֶד (chesed) → Güte / Huld (Bundestreue mitdenken); l3 „Liebe/Güte/Treue" je Kontext
- אֱמֶת / אֱמוּנָה → Treue / Wahrheit / Beständigkeit
- נֶפֶשׁ (nefesch) → Seele (l1); l3 „Leben / Ich / Kehle" je Kontext
- רוּחַ (ruach) → Geist / Atem / Wind je Kontext
- לֵב / לֵבָב → Herz
- תּוֹרָה (Tora) → Weisung (l1); l3 „Weisung / Gebot Gottes"
- מִשְׁפָּט → Recht / Urteil; צֶדֶק / צְדָקָה → Gerechtigkeit
- צַדִּיק → Gerechter; רָשָׁע → Frevler / Gottloser
- שָׁלוֹם → Friede / Heil
- כָּבוֹד → Herrlichkeit / Ehre

## Bilder (in l1 wörtlich erhalten)
- צוּר / סֶלַע → Fels; מָעוֹז → Bollwerk / Zuflucht; מָגֵן → Schild
- קֶרֶן → Horn (Bild für Macht/Stärke — l1 wörtlich „Horn")
- יָד → Hand; זְרוֹעַ → Arm; כָּנָף → Flügel

## Orte / Totenreich / Zeit
- שְׁאוֹל (Scheol) → Scheol (l1) / Totenreich (l3)
- בּוֹר / שַׁחַת → Grube; עוֹלָם → Ewigkeit / „für immer"
- צִיּוֹן → Zion; יְרוּשָׁלִַם → Jerusalem

## Gattungs-/Überschrift-Begriffe
- מִזְמוֹר (Mizmor) → Psalm; שִׁיר → Lied; מַשְׂכִּיל → Maskil; מִכְתָּם → Miktam; תְּפִלָּה → Gebet
- לַמְנַצֵּחַ → „Dem Chormeister" / „Für den Chorleiter"
- לְדָוִד → „von David" / „Ein Psalm Davids" (je nach Konstruktion)
